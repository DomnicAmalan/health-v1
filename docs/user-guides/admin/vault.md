# RustyVault Secrets Management

<!-- TODO: Document RustyVault administration -->

## Overview

RustyVault is a HashiCorp Vault-compatible secrets management service running on port 4117.

## Features

- Secret storage and retrieval
- AppRole authentication
- Userpass authentication
- Token management
- Policy management
- Encryption key rotation

## Admin UI

The RustyVault UI (port 8215) provides a web interface for managing secrets and configuration.
