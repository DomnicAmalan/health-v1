## Data Access

### YottaDB API Endpoints

**Base URL:** `http://localhost:9091/api/v1`

**Get all patients:**
```bash
GET /ehr/patients
```

**Get specific patient:**
```bash
GET /ehr/patients/1  # John Doe
GET /ehr/patients/4  # Maria Garcia
```

**Get patient problems:**
```bash
GET /ehr/patients/1/problems
```

**Get patient medications:**
```bash
GET /ehr/patients/1/medications
```

**Get patient vitals:**
```bash
GET /ehr/patients/1/vitals
```

### Frontend Hooks

**Use in React components:**
```typescript
import { useEhrPatient } from '@/hooks/api/ehr/useEhrPatients';
import { useEhrPatientProblems } from '@/hooks/api/ehr/useEhrProblems';

// In component
const { data: patient } = useEhrPatient('1');
const { data: problems } = useEhrPatientProblems('1');
```
